Artist Statement

During our processing module, we completed sets of code. One of which was a self portrait, and the other was a modification of Dan Shiffman's Bouncing Ball exercise. For the self portrait I wanted to do a simple cartoon-themed picture. Simple, and fun to develop. The Bouncing Ball modification was a trip. Starting out with the base code, I researched and figured out how to apply some more character to the project. I wanted to give the user the ability to modify the sizes of the emerging bubbles. How it works: the user clicks the mouse to release ten bubbles of a random size ranging from the 0 to the size to the user's bubble. The user can use the scroll wheel to increse/decrease the size of the bubbles, therefore increasing the size range of the new bubbles.
